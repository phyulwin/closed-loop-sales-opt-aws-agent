To have GitHub Pages serve the repository root (so `index.html` in the repo root is used), go to your repository's Settings â†’ Pages and set the source to `main` branch and folder `/ (root)`.

To preview locally, you can run the included server:

```powershell

```

Then open http://localhost:3000 in your browser.
