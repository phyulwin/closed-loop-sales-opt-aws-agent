AWS Lambda → runs your agent (compute).

Amazon DynamoDB → stores product info and sales logs.

Amazon CloudWatch → logs agent actions.

(Optional) Amazon API Gateway → to trigger the agent from your frontend.

That covers multiple approved AWS services.
